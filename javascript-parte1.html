🟄 🟄 🟄 🟄 🟄 🟄 🟄 🟄 🟄 🟄 Conceitos Básicos 🟄 🟄 🟄 🟄 🟄 🟄 🟄 🟄 🟄 🟄

Variáveis

As variáveis nada mais são que um espaço na memória do seu computador, onde você pode armazenar qualquer informação. Pode ser um número, uma frase, uma imagem, qualquer coisa. E elas são:
let - São variáveis que PODEM ser ALTERADAS
const - São CONSTANTES que não podem ser alteradas
var - São variáveis que NÃO ESTÃO MAIS EM USO, ou seja, NÃO É PARA USAR MAIS
Exemplo:

let (nome da variável) = (valor da variável)
let quantidadeDeAlunosNaTurma = 30

Obs: Veja que o nome da variável começa com letra minúscula, e quando tem mais de uma palavra, a primeira letra de cada palavra é maiúscula para diferenciá-la (alunosDaTurma). Isso porque não pode haver espaço no nome da variável, nem começar com número ou caracteres especiais. Também não pode haver duas variáveis com o mesmo nome.

String - Textos

Podemos escrever no JS com "aspas duplas" , 'aspas simples' ou `crase` , a diferença é que com as aspas duplas ou simples, não é possível quebrar linha, já com a crase é possível, e além disso, com a crase é possível usar expressões dentro do texto.
"Texto com aspas duplas"
'Texto com aspas simples'
`Texto com crase`

Template Strings ou Templates Literals

As template strings são uma forma de escrever texto usando a crase onde você pode colocar elementos de código dentro, e para isso se usa ${variável}, exemplo:

const idade = 28
const carro = "Fusca"
console.log(`Eu tenho ${idade} anos e meu carro é um ${carro}, estamos no ano de ${2000 + 26}.`)
Com isso será impresso no console: "Eu tenho 28 anos e meu carro é um Fusca, estamos no ano de 2026."

Numbers

Os numeros no JS podem ser inteiros ou decimais (3.14) e podem ser usados para contas ou apenas textos, exemplo:

const idade2 = "28"
const conta = 30 + 20
console.log(`Minha idade ${idade2} e ${conta} reais na conta`)
Com isso é impresso no console: "Minha idade 28 e 50 reais na conta"
Obs: Repare que para fazer uma conta matemática não se usam aspas, isso para o JS entender que você quer fazer a conta e não escrever um texto. E a const "idade2" está com o número dois pois já usamos a variável "idade" para o exemplo anterior, então não podemos repetir o nome da variável.

Null e Undefined

O null é o valor que uma variável tem quando ela não tem nada, ou seja, está sem resposta, exemplo:
let variavel = null
Já o undefined aparece quando uma variável é declarada, mas não tem um valor atribuído, ou seja, ela existe, mas seu valor não foi definido (é como se fosse um erro).

Boolean

O boolean é um tipo de dado que só pode ter dois valores: true (verdadeiro) ou false (falso), ele é muito usado para fazer comparações e tomar decisões no código, exemplo:

const maiorDeIdade = true
const menorDeIdade = false

Functions

As funções são trechos de código que só são executados quando são chamados. Elas são usadas para organizar o código e evitar repetições, e para criar uma função basta usar a palavra function + nome da função + parênteses + chaves, exemplo:

function Cheguei() { console.log("Olá, seja bem-vindo!"); }
Para executar esta função basta usar: Cheguei()

Alteradores de Texto

Essas são funções que permitem alterar o texto de um elemento do HTML.
Text Content - (constante que tá o texto.textContent) // SÓ HTML
Inner Text - (constante que tá o texto.innerText) // LEVA EM CONTA O CSS
Inner HTML - (constante que tá o texto.innerHTML) // TRAZ TUDO (permite usar tags HTML dentro do texto).
Exemplo:

constante que ta o texto.textContent = "Novo texto"

Alteradores de style

Essas são funções que permitem alterar os elementos como se fosse no CSS.
Exemplo: A constante do elemento.style.propriedade CSS (color, fontSize, backgroundColor) = "novo valor"
Por exemplo se eu quiser mudar a cor de um botão que esta em uma constante button basta usar:

button.style.color = "red"
Ou mudar o tamanho da fonte :

button.style.fontSize = "50px"

Obs: Quando se coloca uma propriedade que no CSS se usa o hífen (-) para separar as palavras, no JS se usa a letra maiúscula para separar as palavras, exemplo: background-color (CSS) = backgroundColor (JS).

Math

São funções que permitem fazer cálculos matemáticos, exemplo:
Math.random() - Gera um número aleatório entre 0 e 1
Math.pow() - Calcula a potência de um número (2^2)
Math.sqrt() - Calcula a raiz quadrada de um número (√4)
Math.PI - Retorna o valor de PI
Também há essas outras funções que arredondam números:
Math.floor() - Arredonda um número para baixo
Math.ceil() - Arredonda um número para cima
Math.round() - Arredonda um número para o inteiro mais próximo
Exemplo:

const exemplo1 = Math.pow(2, 2);
const exemplo2 = Math.sqrt(4);
console.log(Math.PI);
console.log(Math.floor(4.9));

Operadores Aritméticos

Também é um método de fazer contas, como o Math, porém, além das operações simples que já sabemos (+, -, /, *) também há:
% - Resto da divisão
** - Potência
++ - Incremento
-- - Decremento
Exemplo:

const exemplo5 = 11 % 2; - 11 / 2 = 5, resto 1
const exemplo6 = 2 ** 2; - É o número ao quadrado
const exemplo7 = 2 ++ 2; - Adiciona +1
const exemplo8 = 2 -- 2; - Subtrai -1

Operadores de Atribuição

É como os operadores aritméticos mas de uma maneira mais simplificada para economizar código, exemplo:

let exemplo = 20
exemplo += 10
O resultado é 30

Dá pra usar todas as operações matemáticas que já vimos acima.

Operadores de Comparação

São operadores que comparam valores, exemplo:
== - Igual
!= - Diferente
=== - Igualdade de VALOR e TIPO
!== - Diferença de VALOR e TIPO
> - Maior que
< - Menor que
>= - Maior ou igual que
<= - Menor ou igual que
Exemplo:

const exemplo1 = 20;
const exemplo2 = "20";
exemplo1 == exemplo2;
true
( São iguais!, mas tipos diferentes )
const exemplo3 = 20;
const exemplo4 = "20";
exemplo3 === exemplo4;
false
( São diferentes! não são do mesmo tipo )

Operadores Lógicos

São operadores que comparam valores, exemplo:
&& - E
|| - Ou
! - Não

& -> E -> Pessoa Exigente / Pessoa Obediente
true && true = true
true && false = false
false && false = false

| -> Ou -> Tanto faz / Filho Espertinho
true || true = true
true || false = true
false || false = false

! -> Negação / Do contra, inverte a resposta
!true = false ou !false = true

Exemplo:

const digitarASenha = false
const digitarOToken = true
const numeroDaConta = true

if(digitarASenha || digitarOToken || numeroDaConta){
console.log("Logado com sucesso")
} else {
console.log("Autenticação FALHOU")
}

Operador type of e delete

typeof - Mostra o tipo de uma variável
delete - Deleta uma variável
Exemplo:

const myNumber = 30
console.log(typeof myNumber)

🟄 🟄 🟄 🟄 🟄 🟄 🟄 🟄 🟄 🟄 Objetos 🟄 🟄 🟄 🟄 🟄 🟄 🟄 🟄 🟄 🟄

São grupos de informações relacionadas a uma variável. Por exemplo, um cadastro de um usuário: onde, ao invés de criar uma variável para cada informação do usuário, como nome, idade, email, etc., podemos criar um objeto e dentro dele colocar todas essas informações. Ao fazer isso, se você der um console.log no objeto, ele vai imprimir todas as informações dele, exemplo:

const usuario1 = {
nome: "Cesar",
idade: 28,
email: "cesar@gmail.com",
}

Obs: Você consegue buscar apenas uma informação específica do objeto se quiser, basta colocar o ponto e a variável desejada, ex: console.log(usuario1.idade). Também dá para adicionar ou consultar mais informações colocando múltiplos pontos, caso exista um objeto dentro do outro, exemplo: console.log(usuario1.endereco.cidade) e assim por diante.

Objetos dentro de Objetos

Você também pode criar objetos dentro de objetos, normalmente para organizar melhor as informações que possuem sub-informações, como por exemplo, um endereço, onde você tem o nome da rua, número, cidade, estado, etc. Isso facilita na hora de buscar informações com o console.log. Para fazer isso, basta abrir uma nova chave após a variável ( endereço: {novas variáveis} ), exemplo:

const usuario2 = {
nome: "Cesar",
idade: 28,
email: "cesar@gmail.com",
endereco: {
rua: " Rua Brasil",
numero: 87,
cidade: "Guarulhos",
estado: "SP", }
}

Obs: Você pode alterar as informações do objeto, mesmo que ele seja criado com const. Basta colocar o nome da variável const + ponto + variável que deseja alterar = "novo valor", exemplo: usuario2.nome = "Cesar Rocha" ou usuario2.endereco.cidade = "São Paulo".

🟄 🟄 🟄 🟄 🟄 🟄 🟄 🟄 🟄 🟄 Array 🟄 🟄 🟄 🟄 🟄 🟄 🟄 🟄 🟄 🟄

Uma array é como se fosse uma box de informações para facilitar a organização de dados relacionados, assim como facilitar posteriormente na busca e edição desses itens dentro da array, e dentro dela é possível colocar qualquer tipo de informação, como números, textos, objetos, outras arrays, etc, e para criar uma array basta usar colchetes [ ] para colocar os elementos, exemplo:

const arrayUsuarios = [
{ nome: "Cesar", idade: 28 },
{ nome: "Lucas", idade: 30 },
{ nome: "Pedro", idade: 29 }
]

Obs: Você consegue editar as informações dentro da array seguindo a mesma lógica que vimos anteriormente, exemplo: arrayUsuarios[0].nome = "Cesar Rocha" (nesse caso a posição do elemento vai dentro dos colchetes [ ] da array);
No array, a ordem dos elementos se inicia no 0, ou seja, o primeiro elemento da array é o 0, o segundo é o 1 e assim por diante.

🟄 🟄 🟄 🟄 🟄 🟄 🟄 🟄 🟄 🟄 If e Else 🟄 🟄 🟄 🟄 🟄 🟄 🟄 🟄 🟄 🟄

O if e o else são estruturas de controle de fluxo que permitem executar diferentes blocos de código com base em uma condição. ( Maior > Menor < ou Igual == )
O if é usado para verificar se uma condição é verdadeira. Você pode usar o if diversas vezes em um código;
Já o else é usado para executar um bloco de código caso a condição do if seja falsa, ou seja, é usado logo após um if, exemplo:

const notaAluno1 = 8
const notaDeCorte = 5
if (notaAluno1 >= notaDeCorte) { console.log("Aluno aprovado!"); }
else { console.log("Aluno reprovado!"); }

Obs: Quando se usa mais de um If pode ter mais que uma condição verdadeira!
Quando for usar a circunstância igual é necessário usar dois sinais de igual (==) para o JS entender que vc quer comparar os valores, e não atribuir um valor a variável.


Else If (juntos)

O else if é usado para verificar se uma condição é verdadeira ou falsa, porém, você pode usar o else if diversas vezes em um código;

Exemplo:

const temperature = 40

if(temperature === 36) {
console.log("Você está saudável")
} else if(temperature > 36 && temperature <= 40){
console.log("Está com febre")
} else if(temperature > 40 ){
console.log("Está com MUITA febre")
}
else {
console.log("Está com hipotermia")
}

Obs: Quando se usa If Else só uma condição pode ser verdadeira !

Operador Ternário

O operador ternário é uma forma mais curta de escrever um if e else, usando o ? como se fosse o if, e : como se fosse o else, exemplo:
? if (se)
: else (se não)
&& (if sem else)

const salary = 5000

// if sem else: Se o salário for menor que 3.000 é Junior, caso contrário não faz nada
salary < 3000 && console.log("Júnior")

// if e else: Se o salário for menor que 3.000 é Junior, se não é Pleno
salary < 3000 ? console.log("Júnior") : console.log("Pleno")

// else if: Se o salário for menor que 3.000 é Junior, se não for menor que 3.000 e for menor que 10.000 é Pleno, se não é Sênior
salary < 3000 ? console.log("Júnior") : salary > 3000 && salary < 10000 ? console.log("Pleno") : console.log("Sênior")

🟄 🟄 🟄 🟄 🟄 🟄 🟄 🟄 🟄 🟄 getElement 🟄 🟄 🟄 🟄 🟄 🟄 🟄 🟄 🟄 🟄

O getElement é uma família de funções do JavaScript que permite acessar elementos do HTML para poder manipulá-los, ou seja, pegar um elemento do HTML e fazer algo com ele usando o JS, como por exemplo, mudar o texto, a cor, a posição, etc. Para acessar o documento do HTML no JS, é utilizado o document. e para buscar o elemento, basta usar document.getElementById("id do elemento"), exemplo:

const titulo = document.getElementById("titulo")

getElementById

O getElementById é uma função que permite pegar um elemento do HTML pelo id (id é único),exemplo:
document.getElementById("id-selecionado")

getElementByClassName

O getElementByClassName é uma função que permite pegar um elemento do HTML pelo class (class é múltiplo),exemplo:
document.getElementsByClassName("class-selecionada")

getElementByTagName

O getElementByTagName é uma função que permite pegar um elemento do HTML pelo tag (tag é múltiplo), nesse caso o getElementByTagName vai pegar todos os elementos da TAG selecionada como p, h1, input, etc.exemplo:
document.getElementsByTagName("tag-selecionada")

getElementByName

O getElementByName é uma função que permite pegar um elemento do HTML pelo name (name é múltiplo), é igual o class mas usando a TAG name exemplo:
document.getElementsByName("nome-selecionado-do-name")

🟄 🟄 🟄 🟄 🟄 🟄 🟄 🟄 🟄 🟄 querySelector 🟄 🟄 🟄 🟄 🟄 🟄 🟄 🟄 🟄 🟄

O querySelector é uma função que permite pegar o primeiro elemento do HTML que corresponda ao selecionado, ele usa o mesmo conseito do CSS para essa seleção, exemplo:
document.querySelector("seleção desejada") a seteção pode ser uma TAG (p, h1, input, etc) ou um ID (#id-selecionado) ou uma CLASS (.class-selecionada) e já que ele pega APENAS o primeiro elemento que corresponde a seleção, também é possível por exemplo filtrar pela TAG e CLASS ou ID, exemplo:
document.querySelector("p.class-selecionada") ou document.querySelector("button.#id-selecionado")
Obs: Eu só consegui usar quando coloquei o código do Java Script depois do body, e abri no navegador.

querySelectorAll

O querySelectorAll é uma função que permite pegar todos os elementos do HTML que correspondam ao selecionado, usando o mesmo princípio do querySelector.

O addEventListener permite adicionar um evento a partir de uma ação específica pré selecionada, como um click ou um change por exemplo:

Exemplo

Constantes usadas no exemplo:
const select = document.querySelector(".selectAel")

🟄 🟄 🟄 🟄 🟄 🟄 🟄 🟄 🟄 🟄 Switch Case 🟄 🟄 🟄 🟄 🟄 🟄 🟄 🟄 🟄 🟄

O switch case é uma estrutura de controle de fluxo que permite executar diferentes blocos de código com base em uma resposta específica ( exemplo: 1, 2, ou 3), diferente do if e else que usa condições ( exemplo: de 1 a 5, de 5 a 10, maior que 10, etc), o switch usa respostas específicas, exemplo:

QUAL DIA DA SEMANA ENTRE SEGUNDA A SEXTA VOCÊ PREFERE? (Digite de 1 a 5)


Resposta:


switch (diaDaSemana.value) {

case "1":
respostaSwitch.innerHTML = "Segunda-feira";
break
case "2":
respostaSwitch.innerHTML = "Terça-feira";
break
case "3":
respostaSwitch.innerHTML = "Quarta-feira";
break
case "4":
respostaSwitch.innerHTML = "Quinta-feira";
break
case "5":
respostaSwitch.innerHTML = "Sexta-feira";
break
default:
respostaSwitch.innerHTML = "Dia inválido"
}

Obs: Eu usei o Switch Case dentro de uma function > addEventListener para eu pegar o que tava escrito no input e mostrar na tela. (Na dúvida, vale conferir).

🟄 🟄 🟄 🟄 🟄 🟄 🟄 🟄 🟄 🟄 Set Time Out e Set Interval 🟄 🟄 🟄 🟄 🟄 🟄 🟄 🟄 🟄 🟄

O setTimeout é uma função que permite executar um bloco de código após um determinado tempo, exemplo: depois de 10 segundos no site executar um alert dando bom dia (uma vez só).
o setInterval é uma função que permite executar um bloco de código a cada determinado tempo, exemplo: a cada 10 segundos no site executar um alert dando bom dia (várias vezes).
Construção:

setTimeout(function () { ---------------> Coloca o que vai ser executado
console.log("Olá, seja bem-vindo!")
}, 10000) ------------------------------> 10000 milissegundos = 10 segundos


setInterval(function () { --------------> Coloca o que vai ser executado
console.log("Olá, seja bem-vindo!")
}, 10000) -----------------------------> 10000 milissegundos = 10 segundos

Obs: Você pode nomear essas funções set com se fosse uma variável, para depois por exemplo usar o clearTimeout ou clearInterval para cancelar a execução da função.
exemplo:

let tempo = setInterval(function () { ---------------> Coloca o que vai ser executado
console.log("Olá, eu sou Tom!")
}, 1000) ------------------------------> 1000 milissegundos = 1 segundos

clearInterval(tempo) --------------------------> Cancela (pausa) a execução da função

🟄 🟄 🟄 🟄 🟄 🟄 🟄 🟄 🟄 🟄 For 🟄 🟄 🟄 🟄 🟄 🟄 🟄 🟄 🟄 🟄

A estrutura de repetição for é usada para repetir um bloco de código um determinado número de vezes.
Obs: É usado uma váriavel para dizer a quantidade de vez que o bloco de código vai ser repetido, normalmente se usa a letra "i" para representar essa variável.
Construção:

for ( let i = 0; i < 11; i++ ) {
console.log (i)
}

Então o i = 0 até chegar em i < 11 vai se repetir (1++ = soma mais 1) até atingir essa regra.

Exemplo prático: Escrever no Console os nomes da lista abaixo:

for (let i = 0; i < user.length; i++) {
console.log(user[i])
}

Obs: O atributo .length é para pegar o tamanho do array, ou seja, quantas pessoas tem na lista.

Break é usado para parar o loop (interromper o código).
Continue é usado para pular uma interação do loop (interromper a interação atual e ir para a próxima).

For of

O for of é usado para iterar (mostrar) os itens de um array, como por exemplo essa lista de nomes que fizemos acima, exemplo:

Construção:

const contactsForof = ['Rodolfo', 'Maria', 'Paulo', 'Aline']

for (const name of contactsForof) {

console.log(name) }

For in

O for in diferente dos anteriores é usado para mostrar os itens de um objeto.

Construção:

for (const key in userForin) {
console.log(key) ----------------> // Mostra as chaves do objeto (name, age, city)
console.log(userForin[key]) ------> // Mostra os valores do objeto (Cesar, 28, São Paulo)
}
Se quisermos pegar a resposta completa do objeto (valores e chaves) fazemos assim:

for (const key in userForin) {
console.log ( key + " : " + userForin[key] ) --------------> // Forma mais simples
console.log (`${key} : ${userForin[key]}`) ----------------> // Mais profissional
}

🟄 🟄 🟄 🟄 🟄 🟄 🟄 🟄 🟄 🟄 While e Do While 🟄 🟄 🟄 🟄 🟄 🟄 🟄 🟄 🟄 🟄

O while é usado para repetir um bloco de código com uma condição, enquanto a condição for verdadeira ele vai repetir a ação, exemplo:

while (condição) {
código que vai rodar
}

Já o do while primeiro executa o código e depois verifica a condição, exemplo:

do {
código que vai rodar
} while (condição);

Obs: Assim como visto anteriormente o break e o continue também podem ser usados aqui.

🟄 🟄 🟄 🟄 🟄 🟄 🟄 🟄 🟄 🟄 For Each 🟄 🟄 🟄 🟄 🟄 🟄 🟄 🟄 🟄 🟄

O forEach utiliza um array interligado a ele para mostrar mais algumas informações que os anteriores que vimos até agora não mostram, como por exemplo o index (posição do item 0, 1, 2, 3, no array), item (são as linhas de informações dentro do array) e array (o array em si), exemplo:

Array e item:

const arrayUsers = [
{ name: 'Rodolfo', age: 33, contact: '(19) 94343-3434' }, ----------> Item
{ name: 'Paulo', age: 21, contact: '(12) 93443-3434' },
{ name: 'Aline', age: 40, contact: '(13) 94566-3434' },
{ name: 'Maria', age: 12, contact: '(14) 94343-3476' },
]


Construção:

array.forEach(element => { -----------------> (=>) Arrow function, a mesma coisa que uma function normal
(código a ser executado)
});


Obs: Se na hora de usar o index para mostrar as posições do array, você não quiser começar com o 0, basta usar (index + 1) no lugar do index.

Exemplo prático:

arrayUsers.forEach((item, index) => {
console.log(`${index + 1} - Nome: ${item.name} - Idade: ${item.age} - Telefone: ${item.contact}`)
});
