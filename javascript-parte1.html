🟄 🟄 🟄 🟄 🟄 🟄 🟄 🟄 🟄 🟄 Variáveis 🟄 🟄 🟄 🟄 🟄 🟄 🟄 🟄 🟄 🟄

As variáveis nada mais é que um espaço na memória do seu computador, onde vc pode armazenar qualquer informação, pode ser um número, uma frase, uma imagem, qualquer coisa E as elas são:
let - São constantes que PODEM ser ALTERADAS
const - São CONSTANTES que não podem ser alteradas
var - São constantes que NÃO ESTÃO MAIS EM USO, ou seja NÃO É PARA USAR MAIS
Exemplo:

let (nome da variável) = (valor da variável)
let quantidadeDeAlunosNaTurma = 30
Obs: Veja que o nome da váriavel começa com letra minúscula, e quando tem mais de uma palavra, a primeira letra de cada palavra é maiúscula para diferencia-la, isso por que não pode haver espaço no nome da variável, nem começar com número ou caracteres especiais. Também não pode haver duas variáveis com o mesmo nome.

String - Textos

Podemos escrever no JS com "aspas duplas" , 'aspas simples' ou `crase` , a diferença é que com as aspas duplas ou simples, não é possível quebrar linha, já com a crase é possível, e além disso, com a crase é possível usar expressões dentro do texto.
"Texto com aspas duplas"
'Texto com aspas simples'
`Texto com crase`

Template Strings ou Templates Literals

As template strings são uma forma de escrever texto usando a crase onde você pode colocar elementos de código dentro, e para isso se usa ${variável}, exemplo:

const idade = 28
const carro = "Fusca"
console.log(`Eu tenho ${idade} anos e meu carro é um ${carro}, estamos no ano de ${2000 + 26}.`)
Com isso será impresso no console: "Eu tenho 28 anos e meu carro é um Fusca, estamos no ano de 2026."

Numbers

Os numeros no JS podem ser inteiros ou decimais (3.14) e podem ser usados para contas ou apenas textos, exemplo:

const idade2 = "28"
const conta = 30 + 20
console.log(`Minha idade ${idade2} e ${conta}`)
Com isso é impresso no console: "Minha idade 28 e 50"
Obs:Repare que para fazer a conta não se usa aspas, para o JS entender que vc quer a conta, e nao o texto, e a const "idade2" está com o dois pois ja usamos a variável "idade" para o exemplo anterior então não podemos repetir o nome da variável.
