getElementByTagName

O getElementByTagName é uma função que permite pegar um elemento do HTML pela tag, exemplo:
document.getElementsByTagName("p")

querySelector

O querySelector é uma função que permite pegar um elemento do HTML por meio de um seletor CSS, exemplo:
document.querySelector("#exemplo1")
document.querySelector(".exemplo2")
document.querySelector("p")
ou com a TAG na frente do seletor:
document.querySelector("p.exemplo2")
ou com ID na frente do seletor:
document.querySelector("p#exemplo1")
