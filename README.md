
# 📚 Meu Primeiro Projeto Web (HTML + CSS + JavaScript) - Tudo que eu Aprendi

Projeto desenvolvido como meu espaço pessoal de estudos em programação, onde organizo anotações, exemplos práticos e conteúdos que venho aprendendo durante minha jornada como desenvolvedor.

O objetivo deste site é funcionar como uma central de aprendizado contínuo, reunindo conceitos, testes, exercícios e práticas de desenvolvimento web utilizando HTML, CSS e JavaScript.

## 🔗 Acesse o projeto

👉 https://cesarrocharl3-blip.github.io/Tudo-que-eu-Aprendi/

## 📖 Sobre o projeto

Este projeto foi criado para registrar e praticar tudo o que venho aprendendo em programação.

Ao invés de apenas consumir conteúdo, decidi criar um ambiente próprio para documentar meus estudos, revisar conceitos e aplicar conhecimentos na prática.

O site funciona como uma espécie de "caderno digital de programação", onde posso:

- Registrar aprendizados
- Testar códigos
- Revisar conceitos
- Organizar estudos
- Evoluir minhas habilidades de desenvolvimento

O principal objetivo deste projeto é acompanhar minha evolução na programação e consolidar conhecimentos através da prática.

Além disso, este repositório também funciona como parte do meu portfólio de desenvolvimento, demonstrando meu processo de aprendizado e dedicação à área de tecnologia.

## 🤖 Nota sobre o desenvolvimento

Este projeto foi desenvolvido por mim como estudante de programação sem qualquer auxilio de I.A já que não faria sentido na questão da prática em si.
