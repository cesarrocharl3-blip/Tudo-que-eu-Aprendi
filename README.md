# 🚀 Meu Primeiro Projeto Web (HTML + CSS + JavaScript)
Estou começando na área de programação.  
Esse é só o começo — bora evoluir!
--
Esse aqui é um projeto que eu desenvolvi enquanto estava aprendendo **HTML e CSS** — colocando na prática tudo que fui estudando nas aulas.

A ideia foi simples:  
👉 criar páginas explicando o que eu aprendi… **usando o que eu aprendi** 😄

---

## 📚 Sobre o Projeto

Esse projeto é basicamente um mini guia interativo com:

- 📄 Página de **HTML**
- 🎨 Página de **CSS**
- 🔗 Navegação entre páginas
- 🧠 Explicações + exemplos práticos

Tudo feito do zero, com foco em aprendizado.

---

## 🧩 O que você vai encontrar aqui

### 🔤 HTML
- Estrutura básica
- Tags de texto, imagem e links
- Listas e tabelas
- Formulários e inputs
- Iframes e vídeos
- Semântica (header, section, etc)

### 🎨 CSS
- Seletores (id, class, etc)
- Cores e fontes
- Box model 
- Display 
- Layout básico
- Estilização geral
- Posicionamento
- Responsividade
