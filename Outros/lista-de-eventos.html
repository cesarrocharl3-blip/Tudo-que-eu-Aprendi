Eventos JavaScript

🖱️ Eventos de Mouse

  • onclick - Clicou no elemento
  • ondblclick - Clicou duas vezes no elemento
  • onmousedown - Pressionou o botão do mouse
  • onmouseup - Soltou o botão do mouse
  • onmouseover - Mouse entrou em cima do elemento
  • onmouseout - Mouse saiu de cima do elemento
  • onmousemove - Mouse está se movendo sobre o elemento
  • onmouseenter - Mouse entrou no elemento (não propaga para filhos)
  • onmouseleave - Mouse saiu do elemento (não propaga para filhos)
  • oncontextmenu - Clicou com o botão direito
  • onwheel - Rodou a roda do mouse

⌨️ Eventos de Teclado

  • onkeydown - Pressionou uma tecla
  • onkeyup - Soltou uma tecla
  • onkeypress - Pressionou e soltou uma tecla (obsoleto)

📋 Eventos de Formulário

  • onfocus - Elemento recebeu foco
  • onblur - Elemento perdeu foco
  • onchange - Valor do campo foi alterado
  • oninput - Valor está sendo digitado em tempo real
  • onsubmit - Formulário foi enviado
  • onreset - Formulário foi resetado
  • onselect - Texto foi selecionado dentro de um input
  • oninvalid - Campo não passou na validação do formulário

🌐 Eventos de Página

  • onload - Página ou elemento terminou de carregar
  • onunload - Usuário está saindo da página
  • onbeforeunload - Antes do usuário sair da página
  • onresize - Janela do navegador foi redimensionada
  • onscroll - Usuário rolou a página
  • onhashchange - A URL mudou após o #
  • onpopstate - Histórico de navegação mudou

🎬 Eventos de Mídia (video/audio)

  • onplay - Mídia começou a reproduzir
  • onpause - Mídia foi pausada
  • onended - Mídia terminou de reproduzir
  • onvolumechange - Volume foi alterado
  • ontimeupdate - Posição atual da mídia mudou
  • onseeking - Usuário está arrastando a barra de progresso
  • onseeked - Usuário terminou de arrastar a barra
  • oncanplay - Mídia pode começar a reproduzir
  • onwaiting - Mídia pausou para carregar mais dados
  • onloadeddata - Dados da mídia foram carregados
  • onratechange - Velocidade de reprodução foi alterada
  • ondurationchange - Duração da mídia mudou
  • onstalled - Navegador tentou buscar dados mas não conseguiu

👆 Eventos de Toque (mobile)

  • ontouchstart - Tocou na tela
  • ontouchend - Removeu o dedo da tela
  • ontouchmove - Está arrastando o dedo na tela
  • ontouchcancel - Toque foi interrompido

📎 Eventos de Drag (arrastar)

  • ondrag - Elemento está sendo arrastado
  • ondragstart - Começou a arrastar o elemento
  • ondragend - Terminou de arrastar
  • ondragover - Elemento arrastado está sobre outro elemento
  • ondragenter - Elemento arrastado entrou em outro elemento
  • ondragleave - Elemento arrastado saiu de outro elemento
  • ondrop - Elemento foi solto sobre outro elemento

📋 Eventos de Clipboard

  • oncopy - Usuário copiou um conteúdo
  • oncut - Usuário recortou um conteúdo
  • onpaste - Usuário colou um conteúdo

🔗 Eventos de Rede

  • ononline - Conexão com internet foi restaurada
  • onoffline - Conexão com internet foi perdida

⚡ Eventos de Animação e Transição CSS

  • onanimationstart - Animação CSS começou
  • onanimationend - Animação CSS terminou
  • onanimationiteration - Animação CSS repetiu
  • ontransitionend - Transição CSS terminou
