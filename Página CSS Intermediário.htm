🟄 🟄 🟄 🟄 🟄 🟄 🟄 🟄 🟄 🟄 Posições 🟄 🟄 🟄 🟄 🟄 🟄 🟄 🟄 🟄 🟄

Existem diversas formas de você conseguir mover seus elementos, uma delas é a função top, rigth, left e botton porém só é possivel usar essas funções em certas ocasiões, normalmente usadas quando se usa a função "positions" determinar a posição desses elementos, essas funções são para:
position: static; (padrão) - É a função padrão que já vem configurado no VSCode, não aceita valores diretos de top, botton, rigth e left
position: fixed; - Fixa o elemento na tela independente da rolagem do site, não se move de lugar
position: sticky; - Ele se atrela a uma margem definida, porém não sai da tela inicial, por exemplo quela barra de menu que fica na tela e ela rola com o conteúdo, porém quando ela vai sair da tela ela não sai e fica sempre na parte superior, isso é devido a essa função "sticky"
position: relactive; - Altera a posição do elemento em relação a posição inicial dele
position: absolute; - Posiciona referente a um elemento, ou a página, se não estiver dentro de um elemento ou se o elemento for "static" (padrão) pisiciona referente a página, agora se estiver dentro de um elemnto não "static", posiciona referente ao elemento

🟄 🟄 🟄 🟄 🟄 🟄 🟄 🟄 🟄 🟄 Pseudo Elementos 🟄 🟄 🟄 🟄 🟄 🟄 🟄 🟄 🟄 🟄

São formas de edição para partes de um elemento, detalhes dentro de um elemento, para escrever se coloca dois, dois pontos (::) após a TAG de identificação exemplos:
Edição da escrita dentro de um input por exemplo: é possível fazer isso >>
fica:input::placeholder { color: blue; font-weight:bold; }
Também é possível alterar só a primeira letra do parágrafo (first-letter), ou a primeira linha (first-line)

Exemplo: p::first-letter { color: blue; font-weight:bold; font-size: 30px;}

E por último o before e after q serve para vc escrever algo antes e após um elemento, por exemplo em um link:
Exemplo de link fica: a::after{ content: "🚀"; } a::before{ content:"🚀" ; }

🟄 🟄 🟄 🟄 🟄 🟄 🟄 🟄 🟄 🟄 Pseudo Classes 🟄 🟄 🟄 🟄 🟄 🟄 🟄 🟄 🟄 🟄

As pseudo classes são parecidas com os pseudo elementos, servem para colocar elementos de edição dentro de TAG's específicas, e diferente do que vimos no pseudo elemento para escrever esse código usa-se apenas um ":" exemplos:
:active → quando o elemento está ativo, ou seja, está sendo pressionado (clicado)
:visited → seleciona o link que foi visitado (tipo os links do google q ficam roxos)
:hover → quando o mouse passe em cima do elemento, pode ser botão, texto, entre outros
:checked → usado para editar os inputs > radio / checkbox / select
:disabled → elemento desabilitado, só parea desabilitar / button / input
:enabled → elemento está habilitado, para habilitar
:focus → é aplicada quando um elemento recebe foco, o que pode ocorrer quando o usuário clica no elemento tipo em um input
:read-only → seleciona input ou textarea que são apenas leitura
:empty → aplica a edição em elemntos vazios
:first-child → para estilizar apenas o primeiro elemento ex: primeira li de uma ul / primeiro span de uma div
:first-of-type → estiliza o primeiro elemento de seu tipo
:last-child → último filho
:last-of-type → estiliza o último elemento de seu tipo
:not() → estiliza todos elementos, exceto o que esta dentro dos parênteses ()
:nth-child() → seleciona o elemento baseado na sua posição, tipo o first e last mas é específico para qual vc quer. Ex: :nth-child(3)
:nth-last-child() → igual o de cima mas conta de trás para frente.

Responsividade nada mais é do que a adaptação da tela grande (PC), para uma tela pequena (celular), pricipalmente na questão questão de tamanhos e proporções dos elementos do seu site, para isso usamos a função @media ()
Com essa função você pode determinar quais elemtos vc quer que mude de acordo com um certo parâmetro, por exemplo a largura da tela em pixel, para isso basta colocar:
@media (max-width:600px) {dentro dos colchetes vc coloca os elementos que mudarção de acordo com esse tamanho (div, p, img...) } Essa função acompanha principalmente duas opções de atributos: @media (max-width:1000px) ou (min-width:1000px)

  • max-width: 500 px - Quando se usa o @mídia com o max, quer dizer que até atingir 500px vai ficar com a config. padrão, se diminuir a partir dos 500px, ou seja a partir de 499px já entra a config. que está dentro da @media
  • (min-width: 500 px) - O "min" ja é o contrário, até os 500px vai manter na config. padrão, a partir disso já altera a configuração para o que está dentro da @media
Exemplo:

Essa div está com essa config. abaixo:
.div5-1 {
width: 1000px;
height: 100px;
background-color: blue; }

DIMINUA O TAMANHO DA JANELA DO SITE PARA VER
A FUNÇÃO @MEDIA EM AÇÃO
Agora se eu aplicar um @midia com max de 1000px quer dizer que ao diminuir menos de 1000px ela trocará para essa config. abaixo:
@media (max-width:1000px) {
.div5-1{
width: 400px;
height: 80px;
background-color: green; } }

Normalmente ao colocar @media se coloca: @media screen and (max-width:1000px) and (orientation: landscape ou portrait)
O screen serve para especificar mais onde será aberto o site, além de portrait ou landscape que é o formato de retrato, ou paisagem (não esquecer de colocar o and)

🟄 🟄 🟄 🟄 🟄 🟄 🟄 🟄 🟄 🟄 🟄 Display Flex e Posicionamentos 🟄 🟄 🟄 🟄 🟄 🟄 🟄 🟄 🟄 🟄 🟄

O display flgit ex é usado para você alinhar um item dentro de um container, ou seja, para alinhar um item dentro de uma div por exemplo.
Exemplo: Para alinhar ->

display: flex;
justify-content: center;
align-items: center;

Por padrão itens dentro de uma div ficam como block , ou seja, ocupam a linha toda horizontalmente (imagem lado esquerdo), ja quando é colocado o display flex os itens ficam como inline , ou seja, ocupam somente o espaço necessário, e ficam um do lado do outro (imagem lado direito)




Flex Direction

Dentro do display flex existe várias funções para te ajudar com a disposição dos itens dentro dele, uma delas é a flex-direction que tem as seguintes opções:
Exemplo de constução: flex-direction: row;

  • row - os itens ficam um do lado do outro, na horizontal (padrão)

  • column - os itens ficam um em baixo do outro, na vertical

  • row-reverse - os itens ficam um do lado do outro, na horizontal, porém na ordem inversa

  • column-reverse - os itens ficam um em baixo do outro, na vertical, porém na ordem inversa

Veja os exemplos aplicados abaixo:




Flex Wrap

O flex-wrap é usado para ter a quebra de linha, caso contrário os itens ficam todos em uma linha só, e quando chegar no final da linha eles vão ficando cada vez menores para caber tudo, com o flex-wrap: wrap; os itens vão para a linha de baixo quando chegar no limite.




Flex Flow

O flex-flow é uma propriedade que combina as propriedades flex-direction e flex-wrap em uma única linha de código, é apenas para economizar linha e não precisar escrever as duas propriedades caso vc esteja usando as duas, exemplo:
flex-flow: row wrap; (a ordem é importante, primeiro a direção e depois o wrap)
flex-flow: column wrap;




Justify Content

O justify-content serve para alinhar os itens dentro do container, exemplo: justify-content: center;. Porém ele tem diversos tipode de alinhamentos diferentes:

  • center - alinha os itens ao centro

  • flex-start - alinha os itens ao início do container

  • flex-end - alinha os itens ao final do container

  • space-between - distribui os itens com espaço igual entre eles

  • space-around - distribui os itens com espaço igual ao redor deles

Obs: Existe o eixo x e y, e usamos principal e secundário, que nada mais é, do que o eixo que os itens estão alinhados como principal, e o outro é o secundário.




Align Items

Já o align-items é para alinhar os itens no eixo secundário, exemplo: align-items: center;, ele tem as seguintes opções de alinhamento:

  • stretch - estica os itens para preencher o container (padrão)

  • center - alinha os itens ao centro

  • flex-start - alinha os itens ao início do container

  • flex-end - alinha os itens ao final do container




Align Content

O align-content é para alinhar as linhas do container, ou seja, quando tem mais de uma linha, ele alinha as linhas, exemplo: align-content: center;, ele tem as seguintes opções de alinhamento:

  • stretch - estica as linhas para preencher o container (padrão)

  • center - alinha as linhas ao centro

  • flex-start - alinha as linhas ao início do container

  • flex-end - alinha as linhas ao final do container

  • space-between - distribui as linhas com espaço igual entre elas

  • space-around - distribui as linhas com espaço igual ao redor dela

Obs: Normalmente para usar o align-content é necessário ter os valor de hight e width definidos e o flex-wrap: wrap;.




Gap

O gap é uma função para colocar um espaçamento entre os itens, exemplo: gap: 20px;, ele tem as seguintes opções de espaçamento:

  • gap: 20px; - coloca um espaçamento de 20px entre os itens

  • row-gap: 20px; - coloca um espaçamento de 20px entre as linhas

  • column-gap: 20px; - coloca um espaçamento de 20px entre as colunas




Flex Basis

O flex-basis é uma função para definir o tamanho inicial dos itens, exemplo: flex-basis: 200px;, ele tem as seguintes opções de tamanho:

  • flex-basis: 200px; - define o tamanho inicial dos itens como 200px

  • flex-basis: auto; - define o tamanho inicial dos itens como o tamanho do conteúdo (padrão)




Flex Grow

O flex-grow é uma função para definir o quanto os itens devem crescer para preencher o espaço disponível, exemplo: flex-grow: 1;, ou seja, o tamanho do item vai aumentar, ou diminuir, de acordo com o espaço disponível.
Como visto na linha de cima o flex-grow tem o número 1, que significa que o item vai crescer de 1 para 1 igualmente entre outros itens, mas caso você coloque a função flex-grow em um item e coloque o número 2, por exemplo: flex-grow: 2;, esse item vai crescer o dobro do que os outros itens, se colocar o número 3, ele vai crescer 3 vezes o tamanho dos outros itens, e assim por diante.




Flex Shrink

O flex-shrink é o inverso do flex-grow, ou seja, ele define o quanto os itens devem encolher para preencher o espaço disponível, exemplo: flex-shrink: 1;, e assim como o flex-grow, o número 1 significa que o item vai encolher de 1 para 1 igualmente entre outros itens, mas caso você coloque 2, 3 em diante, ele vai diminuir mais os itens em relação a o que está 1.
Assim como o flex-grow, o flex-shrink é usado diretamente nos itens.




Atalho -> Flex

O flex é um atalho para os seguintes itens: flex-basis:, flex-grow:, flex-shrink: respectivamente, assim como margin você pode escrever tudo em uma única linha.
Exemplo: flex: 200px 1 1; (flex-basis, flex-grow, flex-shrink)




Order

Com a função order é possível alterar a ordem dos itens dentro do container, exemplo: order: 1;, o item com o número menor vai ficar mais para o início do container, e assim por diante. Você diz em que ordem o item vai ficar dentro do container.
Ao usar o order é necessário colocar em todos os itens do container.




Align Self

O align-self é uma função para alinhar individualmente os itens dentro do container, exemplo: align-self: center;, ele tem as seguintes opções de alinhamento:

  • align-self: auto; - herda o valor do align-items do container (padrão)

  • align-self: center; - alinha o item ao centro

  • align-self: flex-start; - alinha o item ao início do container

  • align-self: flex-end; - alinha o item ao final do container

  • align-self: stretch; - estica o item para preencher o container
