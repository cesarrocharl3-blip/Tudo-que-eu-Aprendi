🟄 🟄 🟄 🟄 🟄 🟄 🟄 🟄 🟄 🟄 TAGs de Imagens 🟄 🟄 🟄 🟄 🟄 🟄 🟄 🟄 🟄 🟄

A primeira forma é salvando uma imagem na pasta que vc criou para o arquivo HTML
Após salvar a imagem aparecerá nos aquivos do VScode
Para pegar essa imagem se usa a TAG "img" e "src", após isso, para acessar a pasta coloque "./" e aparecerá as opções de pasta para buscar a imagem
exemplo: img src="./imagens/logo devclub.png"

Para procurar img em pastas "a cima" depois de colocar "img src" vc coloca " ../ " exemplo: img src="../imagens/logo devclub.png" (depois procura na pasta da mesma forma que antes)

Para mudar o tamanho da altura e largura da imagem usa-se os atributos " width " e " height " dentro da TAG de "img", (usar um ou outro para não distorcer a imagem) Exemplo: " img src="./imagens/logo devclub.png" alt="Logo do DevClub" width="200px" height="200px " "

Para colocar uma descrição na imagem usa-se "alt=" tanto para se a imagem não carregar ter um texto alternativo, como para pessoas com deficiência visual para leitores de tela, Exemplo: img src="./imagens/logo devclub.png" alt="Logo do DevClub"

Existe também a TAG "figure" que serve para criar uma box para colocar uma legenda, e para isso de usa "figcaption"
Exemplo:
" < figure >
< img src="./imagens/logo devclub.png" alt="Logo do DevClub" >
< figcaption > Logo do DevClub < /figcaption >
< /figure "

Há dois tipos de listas, as Ordenadas e Não Ordenadas

A lista ordenada coloca itens numerados, e para criar se usa a TAG " ol " para criar e nos itens coloca-se a TAG " li " exemplo:
< ol>
< li Item 1 < /li>
< li Item 2 < /li>
< li Item 3 < /li>
< /ol>
Exemplo:

  1. Item 1
  2. Item 2
  3. Item 3

Para criar a lista Não Ordenada é a mesma coisa mas se usa a TAG " ul " para criar a lista, e a mesma fica em tópicos e não numerada, exemplo:
Exemplo:

🟄 🟄 🟄 🟄 🟄 🟄 🟄 🟄 TAGs para Elementos Estruturais e Semânticos 🟄 🟄 🟄 🟄 🟄 🟄 🟄 🟄

Há certos elementos q serão principalmente usados na parte do CSS, a "div" e o "span" são exemplos desses elementos

A "div" é uma TAG que cria uma caixa de organização, para depois na hora da edição vc conseguir dividir cada elemento do site

O "span" é uma TAG que cria esse elemento de seleção de caixa dentro de uma palavra ou de uma frase mais específica, também para facilitar na hora da edição

Na ultima atualização do HTML surgiu as TAGs de elementos semânticos, como "header" para o cabeçalho do site, normalmente onde fica o menu de opções "nav" para a barra de navegação, "main" para o conteúdo principal do site, "article" para artigos ou notícias, "section" para seções do site, "aside" para conteúdos laterais "footer" para o rodapé do site. Esses elementos ajudam a organizar melhor o conteúdo e também melhoram o SEO do site, entenda melhor com a imagem abaixo:

Mesmo com a criação desses elementos semânticos, as TAGs "div" são usadas para criar caixas de organização em cada um destes elementos que acabam facilitando a edição do site

Para criar uma tabela se usa a TAG "table" e dentro dela tem as TAGs "tr" para criar as linhas, "th" para criar os títulos das colunas e "td" para criar as células da tabela, exemplo:
"< table>
< tr>
< th> Coluna 1 < /th>
< th> Coluna 2 < /th>
< /tr>
< tr>
< td> Célula 1 < /td>
< td> Célula 2 < /td>
< /tr>
< /table>"
Exemplo:

🟄 🟄 🟄 🟄 🟄 🟄 🟄 🟄 🟄 🟄 TAGs de Vídeo 🟄 🟄 🟄 🟄 🟄 🟄 🟄 🟄 🟄 🟄

Para colocar um vídeo no site se usa a TAG "video" e dentro dela o atributo "src" igual na imagem, mesmo princípio

Para colocar os controles do vídeo se usa o atributo "controls" dentro da TAG "video"

Quando vc coloca os controles, da a opção de download do vídeo, para retirar essa opção se usa o atributo "controlsList="nodownload"

Opção de dar autoplay no vídeo se usa o atributo "autoplay"

E algumas vezes o site bloqueia o video, uma forma de burlar isso é usando o atributo "muted" para tirar o som do vídeo

O tamanho do vídeo se edita usando os atributos "width" e "height" assim como ja vimos anteriormente

Para colocar uma thumb no vídeo se usa o atributo "poster="./pastadaimagem/thumbdovideo.jpg"" dentro da TAG

Exemplo: " video src="./pastadovideo/videoteste.mp4" controls controlsList="nodownload" autoplay muted height="400" poster="./pastadaimagem/thumbdovideo.jpg"> /video>"

Vc pode colocar vários formatos de vídeos para evitar quais quer transtornos com os diferentes navegadores, para isso é só colocar a TAG "source" dentro da TAG "video"
Exemplo:
" < video>
source src="./pastadovideo/videoteste.mp4"
source src="./pastadovideo/videoteste.mkv"
source src="./pastadovideo/videoteste.mp5"
< /video> "

🟄 🟄 🟄 🟄 🟄 🟄 🟄 🟄 🟄 🟄 TAGs de INPUT 🟄 🟄 🟄 🟄 🟄 🟄 🟄 🟄 🟄 🟄

TAGs de "input" são usadas para criar espaços para escrever, como por exemplo em um formulário de endereço ou uma barra de pesquisa, e essa TAG tem váriados tipos diferentes de atributos, como:

  • Auto complete - Para dar a sugestão de completar com seus dados como em um cadastro = autocomplete="street-address ou email ou name"
  • Place holder - Para colocar um texto dentro do campo de escrita para dar a sugestão do que deve ser escrito = placeholder="Texto"
  • Desabled - Para desabilitar o campo de escrita = disabled
  • Read only - Para deixar o campo de escrita somente leitura, ou seja, não pode ser editado = readonly
  • Value - Para colocar um texto dentro do campo de escrita = value="Texto"
  • Auto focus - Para colocar o foco automático no campo de escrita quando a página for carregada (Colocar em um só input pra ele focar)= autofocus

Se vc usar o atributo " type="" " existem váriados tipos de input, como barras de pesquisa, telefone, apenas links (URL) entre outros, como:

  • - color - Para escolher uma cor
  • - date - Para escolher uma data
  • - email - Para escrever um e-mail (tambei usei o "placeholder" ja como demonstração)
  • - number - Para escrever apenas números
  • - password - Para escrever uma senha (os caracteres ficam escondidos)
  • - checkbox - Para criar uma caixa de seleção

- file - Para criar um campo para enviar arquivos
No "file" tem alguns atributos útel como:
mutiple - Para selecionar vários arquivos
accept="tipo do arquivo (.mp3 , .jpg , .mp4)" - Para selecionar um arquivo específivo (Se colocar nas aspas: video/* ou image/* ou audio/* ele pegar todos os tipos de arquivo da mídia escolhida)

O type "radio" consegue fazer esse tipo de seleção única com opções usando "label"

O "label" é usado para criar uma box para colocar o imput dentro como também uma descrição de endereço, ex:
Exemplo:
"< label> Rua:
< input type="text" autocomplete="street-address"> < /input>
< /label> "
Outra opção: Seu input ja estiver criado vc pode usar o " id="chave" " no input para criar um link, e um " for="chave" " no label para linkar eles

🟄 🟄 🟄 🟄 🟄 🟄 🟄 🟄 🟄 🟄 TAGs de FORMS 🟄 🟄 🟄 🟄 🟄 🟄 🟄 🟄 🟄 🟄

A TAG "form" é usada para criar uma box de um formulário, e dentro dela normalmente vc encontra as TAGs de "input" para criar os campos de escrita, de "label" para nomear os inputs, e a TAG "button" para criar um botão de envio do formulário
De acordo com as perguntas desse "form" vc cria os "inputs" necessários e específicos para cada resposta. Exemplo:

Formulário de Cadastro





Atributos úteis para os formulários:

  • Action - Para indicar para onde os dados do formulário serão enviados, exemplo: action="https://www.exemplo.com.br"
  • Method - Para indicar o método de envio dos dados, os mais comuns são "GET" (Envia pela os dados pela URL) e "POST" (Para enviar dados sensíveis), exemplo: method="POST"
  • Enctype - Para indicar o tipo de codificação dos dados, principalmente quando se tem arquivos para enviar, exemplo: enctype="multipart/form-data" (Usado para enviar arquivos)
  • Button type reset - Para criar um botão de resetar o formulário (Apagar todos os dados), exemplo: type="reset"
